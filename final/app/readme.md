# Airfoil Self-Noise Prediction (вариант 13)

Приложение для прогнозирования уровня шума аэродинамического профиля на основе параметров воздушного потока и геометрии профиля.

## Используемые технологии

- Python
- Flask
- scikit-learn
- pandas
- joblib

## Модель

Используется модель `RandomForestRegressor`, обученная на датасете "UCI Airfoil Self-Noise".
Выбрана по причине того, что из всех имеющихся моделей в sklearn показала наилучшие значения ошибки (MAE/RMAE) на конкретном наборе данных.

Предсказываемая величина:
- `scaled-sound-pressure`

## Запуск проекта

Установка зависимостей:

```bash
pip install -r requirements.txt
```
